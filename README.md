# Shop Undo

Undo for the Balatro shop. After a reroll, bring back the **previous top row** of the shop exactly as it was

Press **U** (rebindable) or **right-click the Reroll button** to undo. Press it several times in a row to step further back.

---------------------------------------------------------------------

If you find any bugs, let me know and I'll fix them when I have time.

---------------------------------------------------------------------

## Requirements
- Lovely Injector
- Steamodded

## Installation
1. Install Lovely Injector and Steamodded.
2. Put the `ShopUndo` folder into your Balatro `Mods` folder:
   - **Windows:** `%AppData%/Balatro/Mods/`
   - **macOS:** `~/Library/Application Support/Balatro/Mods/`
   - **Linux (Proton):** `.../steamapps/compatdata/.../pfx/.../AppData/Roaming/Balatro/Mods/`
3. Launch the game. Configure under **Main Menu → Mods → Shop Undo**.
